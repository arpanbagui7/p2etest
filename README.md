## Description

An appointment application

## Installation

```bash
$ yarn install
```

## Setup Env

DATABASE_URL=mongodb://USERNAME:PASSWORD@HOST:PORT/DATABASE
PORT=any 4 digit number

## Prisma Setup

```bash
# create table
$ npx prisma db push

# generate prisma cli
$ npx prisma generate
```

## Running the app

```bash
# build
$ yarn run build

# watch mode
$ yarn run watch

# start app
$ yarn run start

```
